# Project Tracker 🚀

This is a simple task tracking application built using React and TypeScript.  
The goal of this project is to manage tasks by moving them across different stages.

## Features

- Add new tasks
- Move tasks between:
  - To Do
  - In Progress
  - In Review
  - Done
- Delete tasks
- Clean and simple UI

- CSS
